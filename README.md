### Installation & Set Up

1. Install the dependencies

   ```sh
   npm i
   ```

2. Start the dev server

   ```sh
   npm run dev
   ```

### Building and Running for Production

1. Generate a full static production build

   ```sh
   npm run build
   ```

   ```sh
   npm start
   ```
